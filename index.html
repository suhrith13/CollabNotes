How It Works

From the moment you type a character to when your collaborator sees it ‚Äî here's the full journey.

üîê

User Authentication

Users sign up / log in via a REST API. The server issues a JWT token stored in the browser. Every request carries this token so the server knows who you are.

üìÑ

Create / Open a Note

Notes are stored in MongoDB. When you open a note, the client fetches its content via a GET request and renders it in the editor. A unique room ID is assigned per note.

‚ö°

WebSocket Connection

The moment you open a note, your browser establishes a WebSocket connection to the server and joins a "room" specific to that note. This keeps a live, two-way channel open.

‚úçÔ∏è

Typing & Broadcasting

When you type, your client emits a text-change event with the delta (just the change, not the whole doc) to the server. The server broadcasts it to all other users in the same room.

üîÑ

Operational Transform / CRDT

If two users type at the same moment, conflicts are resolved via Quill's built-in delta model (a simple OT approach) ‚Äî ensuring both changes merge correctly without overwriting each other.

üíæ

Auto-save to Database

Every 2 seconds, the server debounces incoming changes and saves the latest document state to MongoDB. If the connection drops and reconnects, you always get the latest saved version.

Technology

The Full Tech Stack

Every tool chosen intentionally. Hover each card to see why it was picked.

‚öõÔ∏è

React.js

Frontend UI

Component-based JavaScript library for building the user interface. Handles routing, state management, and re-renders only what changes ‚Äî keeping the editor fast even while receiving live updates.

WHY IT MATTERS React's state-driven model means when a WebSocket event arrives with new text, only the affected editor component re-renders ‚Äî not the whole page. Crucial for performance in a real-time app.
üü©

Node.js + Express

Backend Server

JavaScript runtime that runs the server. Express adds a minimal web framework for defining REST API routes (login, signup, CRUD for notes). Node's event loop handles thousands of concurrent WebSocket connections without blocking.

WHY IT MATTERS Since both frontend and backend use JavaScript, you share types and logic easily. Node's non-blocking I/O is perfect for real-time apps with many simultaneous connections.
üîå

Socket.IO

Real-time Engine

A library that wraps WebSockets (with fallback to HTTP long-polling). Provides rooms, namespaces, and event-based communication. When Alice types, her delta is emitted to a room ‚Äî only users in that room receive it.

WHY IT MATTERS Raw WebSockets need manual reconnection logic, rooms, and fallbacks. Socket.IO handles all that out of the box ‚Äî letting you focus on the app logic rather than protocol internals.
üçÉ

MongoDB + Mongoose

Database

NoSQL document database. Notes are stored as JSON documents ‚Äî they don't have a rigid schema, so adding fields like "tags" or "version history" later is trivial. Mongoose provides schema validation and query helpers.

WHY IT MATTERS Notes are naturally document-shaped. A note is just a JSON object with content, owner, collaborators, timestamps. MongoDB stores this natively ‚Äî no complex joins needed.
üîë

JWT Authentication

Security

JSON Web Tokens are signed, stateless tokens issued on login. Stored in the browser (HttpOnly cookie or localStorage), they are sent with every API request. The server verifies the signature without needing a database lookup for every request.

WHY IT MATTERS JWTs are stateless ‚Äî no server-side session store needed. This makes your backend horizontally scalable. It's also industry-standard ‚Äî every job description mentions JWT.
üìù

Quill.js Editor

Rich Text

An open-source rich text editor that represents document content as "Deltas" ‚Äî a JSON-based diff format. Instead of sending the whole document on each change, you send just the delta (e.g., "insert 'h' at position 42"). This is the foundation of real-time sync.

WHY IT MATTERS Quill's Delta format is tailor-made for collaboration. Deltas are compact, composable, and conflict-friendly. They integrate directly with Socket.IO events ‚Äî this is the secret sauce of the real-time sync.
üé®

Tailwind CSS

Styling

Utility-first CSS framework. Instead of writing custom CSS classes, you compose styles directly in JSX using utility classes like flex gap-4 rounded-lg. Speeds up UI development dramatically.

WHY IT MATTERS Recruiters notice polished UIs. Tailwind lets a solo developer produce professional-looking interfaces quickly. It's also the most in-demand CSS framework in job postings right now.
